# hirsh

## Description  
A small front-end web project built with HTML, CSS, and JavaScript. It consists of a single page web app (or simple web page) with markup, styling, and dynamic behavior.

## Technologies  
- HTML  
- CSS  
- JavaScript  

## How to Run  
```bash
git clone https://github.com/imenmakhbouche/hirsh.git
cd hirsh
# Open index.html in your browser
```

## Files  
- `index.html` — main HTML file  
- `style.css` — CSS styling  
- `script.js` — JavaScript logic
